Android v23 Users Sample
===================================

Users Sample of the new Android Design library.

- Collapsing Toolbar
- FloatingActionButton
- Snackbar
- Design Patterns

Pre-requisites
--------------

- Android SDK v23
- Android Build Tools v23.0.1
- Android Support Repository for v23.0.1
